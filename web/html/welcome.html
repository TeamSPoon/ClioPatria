Welcome to ClioPatria, the SWI-Prolog Semantic Web Server

This server provides a Web frontend to the SWI-Prolog SemWeb package with support for SPARQL .

About this server

  • Fast in-memory RDF quadruple store.
  • Scalable on-disk RDF triple store.
  • Persistent storage through journal files
  • Partial support for SPARQL 1.1
  • Advanced NLP tools for literal search, stemming, ‘sounds as’, etc.
